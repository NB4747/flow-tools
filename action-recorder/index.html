Web Action Recorder

零成本、无安装、点击即用的网页点击行为录制器

拖拽到书签栏，访问任意网页，一键录制 DOM 点击轨迹。
所有数据留在本地，无需注册，无需付费。

【录制器】拖拽我到书签栏

👆 按住按钮拖到浏览器书签栏，或点击在 本页测试

三步开始使用

1

拖拽到书签栏

将上方红色按钮拖入浏览器的书签栏（Bookmarks Bar）。 如果书签栏未显示，按 Ctrl+Shift+B 打开。

2

访问任意网页

打开你想测试的网页 — 内部系统、Landing Page、H5 活动页均可。 本工具为纯前端脚本，不会向任何服务器发送数据。

3

点击书签开始录制

点击书签栏中的「Web Action Recorder」书签， 右下角弹出悬浮窗，点击页面任意元素即可记录其 CSS 选择器路径。

功能特性

◆ Shadow DOM 隔离 — 悬浮窗样式不会污染宿主页面
◆ 精准选择器 — 自动生成唯一 CSS 路径，优先使用 ID
◆ 点击高亮 — 被记录的元素闪烁红色边框反馈
◆ 一键导出 — 支持复制单条选择器或全部 JSON
◆ 拖拽移动 — 悬浮窗可自由拖动到屏幕任意位置
◆ 零依赖 — 纯原生 ES6 JavaScript，兼容所有现代浏览器
